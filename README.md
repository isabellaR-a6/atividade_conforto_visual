## TechNews — Projeto Teste (Modo Claro e Escuro)
--
 Sobre o projeto
--
O TechNews é um projeto fictício criado apenas para fins de estudo e prática em desenvolvimento web.

A ideia principal foi testar a implementação de modo claro (Light Mode) e modo escuro (Dark Mode) utilizando JavaScript, HTML e CSS.

O conteúdo do jornal é totalmente fictício e foi usado apenas como exemplo visual.

 Funcionalidades
--
Alternar entre Modo Dark e Modo Light

Mudança dinâmica de cores usando JavaScript

Layout simples estilo página de notícia

Botões fixos na tela para troca de tema

 Tecnologias utilizadas
 --

HTML5

CSS3

JavaScript

Objetivo do projeto
--

Este projeto foi desenvolvido como exercício para:

Praticar manipulação do DOM com JavaScript

Trabalhar com eventos de botão

Testar alterações visuais dinâmicas (temas claro e escuro)

Melhorar organização básica de arquivos web

O jornal apresentado é fictício e foi criado somente para testes de interface.
